# Silence Mob! #

## About ##
Silence Mob adds the ability to silence mobs by feeding them any item from the `silencemob:silence_item` item tag (By default the tag contains just Poisonous Potato)

It applies the vanilla silent-tag and the mod can be on a server without the client needing the mod installed.
